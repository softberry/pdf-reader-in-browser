# PDF Reader in Browser

## Work in Progress

This tool is not ready to be used in production.

## How to try:

``yarn``  is selected as package manager. So, 

  1- clone this repo
  
  2- Open a terminal at ``root`` folder 
  
  3- run ``yarn``, to install all dependencies
  
  4- run ``yarn start``

Wait for your default browser to be open on https://localhost:8080

  5- Select/Upload your ``PDF`` file in browser. If your fle can be extracted, you will see the html table.
